# SirenMan


# About SirenMan

SirenMan is a iOS app that let's you play police sirens, use your phone as a microphone and play audio clips from cops in GTA V. The app is built using Swift in Xcode. **This is app is for educational purposes only and should not be used in any illegal manner.**
